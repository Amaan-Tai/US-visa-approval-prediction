# US Visa Approval Prediction

An end-to-end Machine Learning project designed to predict whether a US visa application will be **approved or denied** based on applicant, employer, and job-related attributes.

This project demonstrates how to build a **production-ready ML pipeline** with modular architecture, configuration management, and deployment readiness.

# 🧠 Problem Statement

Visa approval decisions depend on multiple factors such as:

- Applicant profile
- Employer details
- Job category
- Salary level
- Work location
- Visa type

Analyzing these factors manually for large datasets can be inefficient.  
This project uses **machine learning techniques** to learn patterns from historical visa application data and predict approval outcomes.
